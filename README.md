# 🖥️ Meu SO - Simulador de Sistema Operacional em React Native

## 📖 Sobre o Projeto

O **Meu SO** é um simulador de sistema operacional desenvolvido em **React Native utilizando Expo e JavaScript**, 
com o objetivo de reproduzir funcionalidades encontradas em sistemas operacionais reais de forma interativa e intuitiva.

O aplicativo apresenta uma interface semelhante a uma área de trabalho (Desktop), permitindo ao usuário abrir diferentes aplicativos, 
personalizar a aparência do sistema e utilizar ferramentas úteis para o dia a dia.

## ✨ Funcionalidades

### 🖥️ Área de Trabalho (Desktop)

* Interface inspirada em sistemas operacionais convencionais.
* Ícones interativos para abertura dos aplicativos.
* Barra de tarefas localizada na parte inferior da tela.
* Alternância entre tema claro e escuro.

### 🎨 Personalização

* Possibilidade de alterar a imagem de fundo da área de trabalho.
* Diferentes papéis de parede para personalizar a experiência do usuário.

### 🧮 Calculadora

* Calculadora totalmente funcional.
* Realização das quatro operações básicas:

  * Soma
  * Subtração
  * Multiplicação
  * Divisão
* Limpeza de expressões e exibição dinâmica dos resultados.

### 📝 Bloco de Notas

* Criação de novas notas.
* Edição de notas existentes.
* Exclusão de notas.
* Interface simples e prática para anotações rápidas.

### 📅 Calendário

* Calendário funcional integrado ao sistema.
* Visualização organizada das datas.
* Facilita o planejamento e a organização das atividades do usuário.

### ⏰ Relógio

* Exibição do horário oficial brasileiro em tempo real.

### 🎮 Jogos Integrados

#### Sudoku

* Jogo clássico de lógica.
* Interface interativa.
* Possibilidade de preencher e corrigir números durante a partida.

#### Xadrez

* Tabuleiro funcional.
* Movimentação das peças.
* Reprodução das regras básicas do jogo.
  
## 🎯 Objetivo

O projeto foi desenvolvido com o objetivo de aplicar conceitos de desenvolvimento mobile utilizando React Native, explorando:

* Gerenciamento de estado;
* Criação de componentes reutilizáveis;
* Navegação e interação entre telas;
* Manipulação dinâmica da interface;
* Desenvolvimento de aplicações complexas sem a utilização de APIs externas ou bancos de dados.

## 📌 Considerações Finais

O **Meu SO** busca proporcionar uma experiência semelhante a um sistema operacional real, 
reunindo ferramentas úteis, opções de personalização e jogos clássicos em uma única aplicação mobile, 
demonstrando o potencial do React Native para o desenvolvimento de interfaces ricas e interativas.
